.. _sec_glossary:

Glossary
========

.. glossary::

    CPA
        Correlation Power Analysis. An unprofiled attack strategy, where we use
        Pearson's Correlation to evaluate predictions on an intermediate value.

    DPA
        Differential Power Analysis.

    GE
        Guessing Entropy.

    LRA
        Linear Regression Analysis.

    DUT
        Device Under Test.

    SNR
        Signal-to-Noise Ratio.

    SCA
        Side-Channel Analysis.
